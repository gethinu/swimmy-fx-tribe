## Part II: Paul Graham Philosophy (ポール・グレアム)

### Core Principles

1. **Make Something People Want**
   - 人が欲しいものを作れ
   - 技術的に優れていても、欲しい人がいなければ無意味

2. **Do Things That Don't Scale**
   - スケールしないことをせよ
   - 最初は泥臭く、手動で、一人ずつ

3. **Ramen Profitability**
   - ラーメン代稼働
   - 最低限の収益で生き延び、自由を得る

4. **Launch Early, Iterate**
   - 早く出せ、反復せよ
   - 完璧を待つな

5. **Be Relentlessly Resourceful**
   - 執拗に機知を働かせよ
   - 壁にぶつかったら迂回しろ

### Swimmy への適用

- **ラーメン代稼働** = 日次目標 ¥4,546
- **早く出せ** = デモで検証しながら改善
- **スケールしないこと** = 最初は1通貨ペア、小ロット

## Part III: Naval Ravikant Philosophy (ナヴァル・ラヴィカント)

### Core Principles

1. **Specific Knowledge**
   - 特定スキルを磨け
   - 学校では教えられない、あなただけの知識

2. **Leverage (レバレッジ)**
   - **Code** - 限界費用ゼロで複製可能
   - **Media** - 一度作れば何度でも働く
   - **Capital** - お金にお金を稼がせる

3. **Productize Yourself**
   - 自分自身を製品化せよ
   - 自分株式会社

4. **Wealth vs Money**
   - 富 = 寝ている間に稼ぐ資産
   - 金 = 時間を切り売りして得る報酬

5. **Judgment**
   - 判断力が最も希少なスキル
   - 正しい判断は時間をかけて培う

### Swimmy への適用

- **寝ている間に稼ぐ** = 自律トレードbot
- **コードでレバレッジ** = Lisp + Rust
- **特定スキル** = アルゴリズムトレード + AI
