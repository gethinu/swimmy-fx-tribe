## 1. 📊 市場調査プロンプト（煙を探す）

### Reddit検索用
```
site:reddit.com "[ターゲット領域]" "[ペインワード]"

例:
site:reddit.com "concert hotel" "frustration"
site:reddit.com "pokemon card" "restock notification"
```

### NotebookLM → Thinking AI用（事実抽出）
```
【NotebookLMへの指示】
すべてのソースに基づき、以下を客観的に抽出してください：
1. 現在の市場状況と数値データ
2. 顕在化している課題
3. 競合の強みと弱み
4. 未解決のペイン

主観的な解釈は不要です。出力できる限りの情報を徹底的に出力してください。
```

## 2. 🚀 戦略立案プロンプト

### Thinkingモード起動用（GPT-o1 / Gemini 2.0）
```
【役割】
あなたは[領域]を熟知した最高レベルの事業戦略家です。

【コンテキスト】
[NotebookLMで抽出した事実をここに貼り付け]

【タスク】
上記の事実に基づき、[目標]を達成するための戦略を立案してください。

【思考プロセス指示】
- 直感で回答せず、データを元に論理的に推論
- 複数のシナリオを検証してから結論
- 「なぜそう言えるか」の根拠を明示
```

### 未来予測プロンプト
```
現在の進捗ペースと発生している小さな摩擦から、3ヶ月後の状態をシミュレーションしてください。

出力形式：
1. 成功確率（パーセント）
2. 失敗するとしたら「何が致命的な原因」になるか
3. 今すぐ手を打つべきアクション
```

## 3. 💻 開発プロンプト（Vibe Coding）

### Cursor / AI Copilot用
```
[機能の説明]を実装してください。

技術スタック: [言語/フレームワーク]
制約:
- シンプルに保つ
- エラーハンドリングを含める
- コメントは日本語で

最初にファイル構成を提案してから実装してください。
```

### マニュアル自動生成用（動画→テキスト）
```
この動画から操作マニュアルを作成してください。

出力形式：
1. タイトル
2. 前提条件
3. 手順（スクリーンショット付き）
4. 注意点
5. トラブルシューティング
```

## 4. 📺 TikTok/YouTube戦略プロンプト

### 動画企画プロンプト
```
[ジャンル]の動画で、直近30日間でバズった投稿を分析してください。

分析項目：
1. Hook（最初の3秒）のパターン
2. 構成（起承転結）
3. サムネイル/カバーの特徴
4. コメント欄で多い反応
5. 再現可能なテンプレート
```

### TTPテンプレート
```
【Hook】: [0-3秒] [注意を引くフレーズ]
【問題提起】: [3-10秒] [視聴者のペイン]
【解決策】: [10-40秒] [商品/サービス紹介]
【CTA】: [最後3秒] [次のアクション]
```
