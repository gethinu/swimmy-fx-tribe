## 🎬 Project 1: AURA-CAST (部屋診断YouTube)

### コンセプト
AURA-MAPのYouTube版。視聴者の部屋写真を募集し、AI風水診断を動画で解説。

### 収益モデル
| 収益源 | 方法 |
|--------|------|
| YouTube広告 | 再生回数 |
| 楽天アフィリエイト | 「この鏡を置くと金運UP」→ 商品リンク |
| 有料診断 | 個人向け詳細診断は有料（AURA-MAP連携） |

### フォーマット
```
タイトル: 【開運】この部屋、運気下げてます...風水師AIが診断
尺: 10-15分
構成:
1. Before写真（視聴者投稿）
2. AI診断結果
3. 奥様の解説（運気の解説）
4. 改善提案（アフィリ商品紹介）
5. After予想（AI生成）
```

### 差別化
- 顔出し不要（部屋写真メイン）
- AI生成でサムネ/BGM作成 → 編集負担軽減
- AURA-MAPとの相互送客

## 🎬 Project 2: DEAL-RADAR (お得情報チャンネル)

### コンセプト
Product Sniperの検知結果をYouTube Shorts/TikTokで速報。

### 収益モデル
| 収益源 | 方法 |
|--------|------|
| YouTube広告 | ショート動画は量産可能 |
| 楽天アフィリエイト | 概要欄にリンク |
| 有料Discord | 詳細情報はDiscordで |

### フォーマット
```
タイトル: 【速報】Sony α7IV 過去最安！今すぐ買え！
尺: 15-60秒
構成:
1. 商品画像
2. 価格推移グラフ（自動生成）
3. 「今買うべき理由」3行
4. 「概要欄からどうぞ」
```

### 自動化ポイント
- Product Sniperが検知 → 自動で動画素材生成
- 奥様はナレーション録音のみ（後から一括）
- アップロードも予約投稿で半自動

## 🎬 Project 3: ORACLE-LIVE (ライブ抽選会)

### コンセプト
Oshi-Yadoと連携。ライブ遠征の宿を「抽選で1名に無料プレゼント」するYouTubeライブ。

### 収益モデル
| 収益源 | 方法 |
|--------|------|
| YouTube広告 | ライブ視聴 |
| スーパーチャット | ファンからの投げ銭 |
| 楽天/じゃらんアフィリ | 外れた人もリンクから予約 |

### フォーマット
```
タイトル: 【抽選LIVE】Snow Manドームツアー！会場横ホテル1泊プレゼント！
尺: 30分
構成:
1. 対象ライブの紹介
2. 参加条件説明（チャンネル登録+コメント）
3. 雑談/推し語り
4. 抽選発表
5. 外れた人向けに「お得な予約方法」紹介
```

### 差別化
- ライブ配信は「同接」でアルゴリズム有利
- ファンコミュニティ形成 → 継続視聴

## 🎬 Project 4: LIFE-HACK-LAB (生活改善実験)

### コンセプト
Naval的「時間の自由」を追求する実験チャンネル。

### 収益モデル
| 収益源 | 方法 |
|--------|------|
| YouTube広告 | 長尺動画 |
| Amazon/楽天アフィリ | 紹介商品 |
| Notion/ツール紹介 | アフィリエイト |

### 動画案
```
「【検証】全自動洗濯乾燥機で週10時間節約できるか」
「【実験】食洗機 vs 手洗い 1年間の時給換算」
「【比較】電動昇降デスク3種 腰痛持ちが選ぶNo.1」
「【解説】副業サラリーマンの1日ルーティン」
```

### ポイント
- 「時間をお金で買う」テーマはNaval思想と合致
- 商品レビュー系はアフィリと相性◎
- SEキャリアの知見も活かせる
