# Swimmy Daily Review Workflow

This workflow enables continuous AI collaboration between Swimmy and Opus.

## Steps

1. **Read Daily Handoff**
   - View the handoff file: `/home/swimmy/swimmy/.opus/daily_handoff.md`
   - Understand today's trading results and any issues

2. **Read Recent Log** (if needed)
   - Check the most recent log in: `/home/swimmy/swimmy/doc/`
   - Look for patterns, warnings, or anomalies

3. **Analyze Performance**
   - Compare actual vs expected PnL
   - Evaluate strategy effectiveness
   - Identify issues and root causes

4. **Provide Recommendations**
   - Suggest specific code changes if needed
   - Recommend parameter adjustments
   - Propose new features or improvements

5. **Update Handoff Document**
   - Add Opus's analysis and recommendations to the handoff
   - This will be read by Swimmy/User in the next session

## Quick Command

User can simply say: `/daily-review` or "review swimmy" to trigger this workflow.
